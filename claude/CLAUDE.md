## ランタイム・パッケージ管理

| ツール | 用途 |
|--------|------|
| **mise** | ランタイムバージョン管理（Node.js, Python 等）。設定は `~/.config/mise/config.toml` |
| **uv** | Python パッケージ管理・スクリプト実行。PEP 723 インラインメタデータ付きスクリプトは `uv run` で実行する（`python3` で直接実行しない） |
